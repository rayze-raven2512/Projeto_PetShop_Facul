# Mundo Pet - TheoShop (Projeto – Fase 1 e Fase 2)

Este projeto consiste na implementação e evolução do front-end do sistema web de e-commerce do petshop Mundo Pet - TheoShop, passando de uma estrutura base em HTML (Fase 1) para uma aplicação dinâmica e responsiva (Fase 2), conforme requisitos da disciplina.

## Tecnologias Utilizadas

- HTML5: Estrutura do site  
- CSS3: Estilização e tema Dark  
- Bootstrap 5: Layout responsivo e componentes visuais  
- JavaScript: Interatividade e cálculos dinâmicos  

## Estrutura e Funcionalidades da Página

A estrutura do site é dividida em:

### Cabeçalho `<header>`

Contém o nome do PetShop e a navegação principal.  
Exibe aviso dinâmico informando se a loja está Aberta ou Fechada.

### Corpo Principal `<main>`

**Visualização de Produtos:**  
3 categorias (Acessórios, Rações e Higiene).  
Cada item contém imagem, descrição e valor.

**Visualização de Serviços:**  
Banho e Tosa (com e sem tele-busca).  
Informações de valor e descrição.

**Carrossel de Imagens:**  
Banner rotativo com promoções e serviços.

**Formulários:**  
Cadastro de Cliente  
Cadastro de Pet  
Agendamento de Serviços, com cálculo automático do valor total

### Rodapé `<footer>`

Informações de contato e endereço.  
Dados de autoria do projeto.  
Widget VLibras para acessibilidade.
